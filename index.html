About Me

I am a Web Developer with approximately 3 years of professional development experience, primarily in the insurance and finance industries, using ASP.NET web forms. During this period, I have successfully automated organizational processes, produced crisp reports and enhanced overall web application functionality under Agile development cycles. Recently, I have been involved in General Assembly's flagship, 600+ hour, web development immersive program. Following a strong work ethic and a tremendous amount of commitment, I have been able to cultivate sound knowledge of the programming languages, JavaScript and Ruby, and development frameworks, Ruby on Rails and Express. It is my desire to utilize such knowledge to add value to the development initiatives of a progressive firm.

Therefore, at a more philanthropic level, I aim to utilize my skills in development to provide a meaningful platform for the downtrodden and destitute to share their voices through the internet.

The Bird Tracker



A full stack web application built using HTML, CSS and JavaScript with a Ruby on Rails API and PostgreSQL database. Features include a sightings table displaying recorded bird sightings. A signed in user has the ability to create a new bird sighting and update/delete user-owned sightings. Data logged can be exported to excel so that statistical analysis, including the distribution of birds in a given environment as a correlated variable with time, can be performed and used to determine critical factors on bird conditions, including vulnerability and extinction possibilities.
