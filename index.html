About Me

I am a Web Developer with approximately 3 years of professional development experience, primarily in the insurance and finance industries, using ASP.NET web forms. During this period, I have successfully automated organizational processes, produced crisp reports and enhanced overall web application functionality under Agile development cycles.

Recently, I have been involved in General Assembly's 12 week, 500+ hour, Web Development Immersive program where I have demonstrated proficiency in the programming languages, JavaScript and Ruby, and development frameworks, Ruby on Rails and Express. It is my desire to utilize my skills in web development to add value to the development initiatives of a progressive firm.

Hailing from Kenya, I bear witness to abject poverty and the lack of opportunity to many individuals in society. Therefore, at a more philanthropic level, I aim to utilize my skills in development to provide a meaningful platform for the downtrodden and destitute to share their voices through the internet.

The Bird Tracker



A full stack web application built using HTML, CSS and JavaScript with a Ruby on Rails API and PostgreSQL database. Features include a sightings table displaying recorded bird sightings. A signed in user has the ability to create a new bird sighting and modify existing user-owned sightings. Data logged can be exported to excel so that statistical analysis, including the distribution of birds in a given environment as a correlated variable with time, can be performed and used to determine critical factors on bird conditions, including vulnerability and extinction possibilities.

Hello Kenyan



A peace portal built using the ASP.NET web forms framework. HTML, CSS and JavaScript is used for front-end development and event handling.
